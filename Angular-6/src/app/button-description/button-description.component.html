Button

The button tag defines a clickable button. Inside a button tag element you can put content, like text or images. This is the difference between this element and buttons created with the input element.

<input type="button" class = "book-button green" value = "Book" (click)="primaryclick()">

<input type="button" class = "book-button yellow" value = "Partially Available" (click)="partialclick()">

<input type="button" class = "book-button red" value = "Not Available" (click)="notavailableclick()">
